## 11. Notes for Dev / Art Team

- All asset IDs are placeholders (`rbxassetid://000000XXX`) in `AssetConfig.lua` — replace once uploaded to Roblox
- Clothing items that use UGC should be listed in `AssetConfig.Items` with their UGC catalog asset ID
- Textures applied to Roblox parts use `SurfaceAppearance` — provide Albedo, Normal, Roughness, Metalness maps where needed
- All looping ambient SFX should be tested with `SoundService.RespectFilteringEnabled = true`
- BGM uses `Sound` objects inside zone folder; switch via `TweenService` for smooth crossfade on zone transition
- Animation IDs go into `AssetConfig.Weapons[id].animationId` and player rig's `Animator`
- Portrait illustrations: 512×512 minimum, transparent background, consistent art style across all characters
- Loading screen images: 1920×1080 minimum, exported at Roblox's recommended format
- Zone maps (Peta buyable item): can be a 2D image displayed in a ScreenGui frame — no 3D model needed
